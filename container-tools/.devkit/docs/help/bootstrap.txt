usage: bootstrap [-h] --template TEMPLATE [--templates-root TEMPLATES_ROOT]
                 [--output-dir OUTPUT_DIR] [--args [ARGS ...]]
                 [--devkit-log-file DEVKIT_LOG_FILE]

Bootstraps a new project from a template.

options:
  -h, --help            Show this help message and exit.
  --template TEMPLATE   The name of the template to use.
  --templates-root TEMPLATES_ROOT
                        The root directory for templates lookup.
  --output-dir OUTPUT_DIR
                        The directory where the project will be bootstrapped. Defaults to current directory.
  --args [ARGS ...]     A list of key=value pairs for templating.
  --devkit-log-file DEVKIT_LOG_FILE
                        Path to a file for logging. If not specified, logs to stderr.
